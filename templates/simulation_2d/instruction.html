1. Модель Изинга

"Двухмерная модель Изинга исследуемая методом Монте-Карло", далее по тексту - проект, расположен во вкладке "Симулятор". Рабочее пространство разделено на отдельные взаимозависимые модули, далее по тексту - область: "1.Симуляционная модель", "2.Управление симуляцией", "3.Настройка параметров", "4.График" и "5.Выгрузка данных" соответственно. 

1.1 В области(Симуляционная модель) из параметров имеется поле "Размерность" соответствующее 2^n степени. Для удобства использования размерность ограничена с 16х16 до 512х512  пикселей (высота = ширине). Размерность сетки не сбрасывается с кнопок в области управления симуляцией. Во время работы сохраняется несколько измерений, включая энергию тока и намагничивание, их усредненные значения и колебания. Значение тока каждого отображается под самой моделью соответственно.

1.2. Во области(Управление симуляцией) имеются 5 кнопок: Старт - запуск симуляции с выбранными параметрами Шаг - приостановление симуляции, при повторном нажатии совершает шаг на 1 сек. Не зависит от выбранного временного интервала. Сброс - обнуление времени, данных графика и данных для скачивания "на лету" без остановки симуляции. Параметры "темпиратура", "поле", "временной интервал" остаются неизменными. Рестарт модели - Перезапускает симуляцию "на лету", обнуляет все параметры и данные для скачивания. 

1.3.  В область(Настройка параметров) находятся регулируемые параметры, напрямую влиящие на поведение модели. Справа, возле каждого наименования расположено точное значение переменной, которое можно регулировать ползунком. Для более точной корректировки, под каждым ползунком имеется поле ввода с соответствующей подписью. Для копирования имеющегося значения с последующей, более точной корректировкой необходимо нажать {ЛКМ} и оно автоматически переместится в поле ввода. После ввода своего значения в поле для применения изменений нужно нажать {Enter}.При вводе значения, по любым причинам неподходящего, оно округляется до приемлемого, либо не изменяется если округлено до изначально установленного. Для изменения доступны следующие параметры:

Температура – стандартная температура, t критическая = 2.26918531421; 
Поле – внешнее магнитное поле, Н; 
Временной интервал(шаг) – симуляция может быть ограничена графикой, это определяет количество разверток между рисунками. 

Также представлены 2 оптимальных для данной реализации алгоритма(METROPOLIS и WOLFF) взаимодействия и работы с этой симуляцией. 

1.4  В области(График) можете выбрать 1 из 4 типов графика, соответствующих зависимостям от собственных параметров указанных в названии типа. Примечание: график может замедлить симуляцию на долгое время. Чтобы ускорить симуляцию, в «тип графика» выберите «нет». Сбор данных продолжается.


2. Программное обеспечение

Все компоненты проекта распространяются с согласием свободного программного обсепечения, в соответствии с лицензионным соглашением Apache 2.0. Проект разрабатывался и тестировался на движке WebKit. Корректность  работы на стороннем программном обеспечении не гарантирована, что уведомляет об использовании Google Chrome, Opera, либо Safari актуальной версии.
